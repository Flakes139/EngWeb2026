### Resumo:

Criar uma estrutura web para navegação de um dataset sobre reparações de automóveis.
A página inicial deve apresentar as categorias de dados disponíveis para consulta.
As categorias incluem reparações realizadas, tipos de intervenções e marcas/modelos de viaturas.
Cada categoria tem uma página de listagem, e cada item da lista possui uma página de detalhe própria.
